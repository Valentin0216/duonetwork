Ubuntu - DuoNetwork


---- 1. lépés ----

A meglévő csomagok frissítése, illetve telepítése:

sudo apt update
sudo apt upgrade

---- 2. lépés ----

Statikus IP cím beállítása:

nano /etc/netplan/50-cloud-init.yaml

network:
  version: 2
  ethernets:
    enp0s3:
	dhcp4: no
	addresses:
	   - 172.16.10.3/25
	routes:
	  - to: 0.0.0.0/0
	    via: 172.16.10.1
	nameservers:
	  addresses: [172.16.10.3, 8.8.8.8]

Mentés és kilépés után:
sudo netplan apply

---- 3. lépés ----

A szükséges csomagok letöltése, telepítése:

sudo apt install ufw - Tűzfal
sudo apt install apache2 - Webszerver üzemeltetéséhez szükséges
sudo apt install samba - Samba megosztáshoz szükséges
sudo apt install cups - Nyomtató megosztáshoz szükséges

---- 4. lépés ----
Protokollok engedélyezése tűzfalon keresztül:

sudo ufw allow 'Apache Full' - Ez engedélyezi a HTTP,HTTPS protokollokat
sudo ufw allow samba - Samba működéséhez engedélyezi a protokollokat

---- 5. lépés ----
Csináljunk egy új weboldalt:

sudo cp sudo cp /etc/apache2/sites-available/000-default.conf /etc/apache2/sites-available/duonetwork.conf

sudo nano /etc/apache2/sites-available/duonetwork.conf

<VirtualHost *:80>
    ServerName DuoNetwork.net
    DocumentRoot /var/www/duonetwork
    ErrorLog ${APACHE_LOG_DIR}/error.log
    CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

Mentés, Kilépés után:

sudo mkdir /var/www/duonetwork
sudo chown -R duonetwork:duonetwork /var/www/duonetwork
sudo a2ensite duonetwork.conf

Ezek után el kellene érni a 172.16.10.3 IP címmel a weboldalunkat ( el is érjük )

---- 6. lépés ----

Nyomtató, mappa megosztás Samba segítségével:

sudo nano /etc/samba/smb.conf

[Share]
  comment = Shared Folder
  path = /srv/samba/share
  browseable = yes
  writable = yes
  guest ok = yes
  create mask = 0644
  directory mask = 0755

[Printers]
  comment = All Printers
  browseable = yes
  path = /var/spool/samba
  printable = yes
  guest ok = yes
  use client driver = yes

Mentés, Kilépés után:

sudo chmod -R 777 /srv/samba/share
sudo chown -R duonetwork:duonetwork /srv/samba/share
sudo systemctl restart smbd
